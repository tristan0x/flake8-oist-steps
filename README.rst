Flake8 CNS-OIST plugin
======================
Check usage of STEPS subcellular simulator Python API.

Install
-------
Install with pip::

    $ pip install flake8-oist-steps

Requirements
------------
- Python 2.7, 3.5, 3.6, 3.7
- flake8

Usage
-----

```bash
$ flake8 my-steps-script.py
```
